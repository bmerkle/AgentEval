## Troubleshooting Guide

### Problem: Build fails with errors
**Solution**: Fix source code errors and rebuild

### Problem: No XML files generated
**Solution**: Check that `<GenerateDocumentationFile>true</GenerateDocumentationFile>` is in .csproj

### Problem: docfx command not found
**Solution**: Install with `dotnet tool install -g docfx`

### Problem: No API YAML files generated
**Solution**: Check docfx metadata output for errors, verify XML files exist

### Problem: HTML build shows warnings
**Solution**: These are often non-blocking; check if HTML was still generated
